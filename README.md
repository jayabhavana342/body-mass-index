# body-mass-index

The body mass index (BMI) is calculated as follows:

1. BMI<sub>Imperial</sub> = (weight in pounds) * 703 / (height in inches)<sup>2</sup>
2. BMI<sub>Metric</sub> = (weight in kilograms) * 703 / (height in inches)<sup>2</sup>
